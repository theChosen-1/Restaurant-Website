# Restaurant Website

A restaurant landing page built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Live Demo

[View Live Site](https://thechosen-1.github.io/Restaurant-Website/)

## About

This project is a single-page restaurant website for "Zaiqa" - featuring authentic Pakistani cuisine. The page is dynamically generated using JavaScript DOM manipulation, showcasing dishes like Lahori Biryani, Karachi Nihari, and Shahi Kheer.

## Features

- Dynamic content rendering using vanilla JavaScript
- Tab-based navigation (Home, Menu, Contact)
- Webpack bundling with HTML and CSS loaders
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Webpack

## What I Learned

- DOM manipulation with `createElement()`, `appendChild()`, and `textContent`
- ES6 module imports/exports
- Webpack configuration for bundling JavaScript and CSS
- Organizing code into separate modules for different page sections

## Getting Started

```bash
# Clone the repository
git clone https://github.com/inshaal81/Restaurant-Website.git

# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
